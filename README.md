# Indian Cinema DBMS - Backend
## **DBMS Course Project**

A backend system for managing Indian cinema data, including movies, producers, genres, and box-office performance. Designed to support AI-driven analytics and revenue prediction.

## Features

- CRUD operations for Movies, Producers, Genres, and Box Office data  
- Flexible search and filtering capabilities  
- Analytics endpoints using stored procedures for performance insights  
- Complex queries to uncover trends and patterns  
- Triggers for data validation and audit logging  
- RESTful API for seamless integration with frontend or AI modules  

## AI & Analytics Focus

- Structured data ready for machine learning models  
- Supports predictive analysis of movie box-office revenue  
- Provides insights for producers, investors, and distributors  

## Tech Stack

- Python 3.10+  
- FastAPI  
- MySQL 8.0+  
- mysql-connector-python  

## Prerequisites

- Python 3.10 or higher  
- MySQL 8.0 or higher  
- pip (Python package manager)  

## Installation

1. Clone the repository:  
```bash
git clone <your-repo-url>
cd indian-cinema-dbms-backend
````

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Configure your MySQL connection in `config.py`.
4. Start the FastAPI server:

```bash
uvicorn main:app --reload
```
